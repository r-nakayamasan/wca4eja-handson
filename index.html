VSCodeでwCAのインストールをするためのガイド

ここでは、IDE VSCodeで必要なソフトウェアツールのインストール方法について説明します。
Watson Code Assistant (wCA)

Last updated: Nov 13th, 2024

環境設定

1. Java のインストール

こちらのリンクを使って Java21 をインストールしてください:

  • Download Java for MacOS - Arm64
  • Download Java for MacOS - x86
  • Download Java for Windows

上記はすべて圧縮ファイルなので、お使いのローカル環境の任意フォルダで解凍する必要があります。

  • 以下のコマンドで Java がインストールされているかチェックできます:
java --version
  • Javaをインストールした後に環境設定を行います。PATH変数にJavaを追加し、JAVA_HOME環境変数を設定します。

  • Mac:

  • .zshrc or .bash_profile を開きます

    bash nano ~/.zshrc - 以下の行を追記します。 bash export JAVA_HOME=/Library/Java/JavaVirtualMachines/<java version>/Contents/Home bash export PATH=$JAVA_HOME/bin:$PATH - ファイルを保存して終了します（CTRL + X を押し、次に Y を押し、Enter を押します）。 - シェル構成をリロードすると変更が有効になります。 bash source ~/.zshrc - 次のコマンドで JAVA_HOME を確認します: bash echo $JAVA_HOME - Windows: - Windowsの検索バーから環境変数を開きます（検索バーで「環境変数の編集」を検索します）

  • 環境変数から、JAVA_HOME変数を設定します。（JAVA_HOMEが設定されていない場合は新規をクリックし、既存のJAVA_HOMEを変更する場合は編集をクリックし、前のステップでインストールしたJavaを指定します）:

bash JAVA_HOME= C:\Program Files\Java\jdk-21

  • 環境変数を使用して、PATHにJavaを追加します:

bash %JAVA_HOME%\bin

2. Maven のインストール

  • Windows
    • Mavenの公式ウェブサイトを: Maven のダウンロードページ
    • 「Files」の下にあるバイナリZIPアーカイブのリンクをクリックします。 (e.g., apache-maven-x.x.x-bin.zip).
    • ZIPファイルを任意の場所に解凍します。 e.g., C:\Apache\maven.
    • 環境変数を使用して MAVEN_HOME 変数の設定を行います。: bash MAVEN_HOME= <path-to-folder>\maven\apache-maven-3.9.9-bin\apache-maven-3.9.9
    • 環境変数を使用して、PATHにMavenを追加します: bash <path-to-folder>\maven\apache-maven-3.9.9-bin\apache-maven-3.9.9\bin
  • Mac
  • homebrewを使ってMavenをインストールします。 bash brew install maven
  • 以下のコマンドを入力し、Mavenが正しくインストールされているか確認してください。: bash mvn --version

4. wCA API Key

APIキーはIBM社員が提供します。IBM社員にご連絡ください。

5. WCA 拡張モジュールのダウンロード

Marketplaceからwatsonx Code Assistant拡張機能をダウンロードします

「インストール」をクリックします。

そうすると、プロダクト・ページが表示されます。

6. WCAへログイン

ステップ5で拡張機能をインストールした後に、以下の手順で拡張機能にログインしてください。:

  • Login with WCA4EJ API Key at the bottom left corner of VSCode. After successfully signed in, the number indicator should be gone.
